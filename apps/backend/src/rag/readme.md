## 2. 产品说明：智能助手 Pro (Assistant Pro)

"智能助手 Pro" 是智流科技推出的核心企业级 AI 生产力工具。

### 2.1 核心功能

- **文档解析**：支持 PDF, Word, Excel, Markdown 格式，最大单文件支持 50MB。
- **多模态处理**：支持图片中的文字提取 (OCR)，暂不支持视频内容分析。
- **私有化部署**：仅企业版支持本地私有化部署，标准版使用云端 API。
- **数据安全性**：所有云端数据均经过 AES-256 加密，承诺不用于模型训练。

### 2.2 支持语言

- 支持中文、英文、日文、法文、德文。
- 小语种（如泰语、越南语）支持处于测试阶段，准确率约为 85%。

## 4. API 开发指南

### 4.1 认证方式

所有 API 请求需在 Header 中携带 `Authorization: Bearer <YOUR_API_KEY>`。

### 4.2 速率限制 (Rate Limit)

- 个人免费版：每秒 1 次请求 (1 QPS)。
- 专业版：每秒 10 次请求 (10 QPS)。
- 超过限制将返回 HTTP 状态码 `429 Too Many Requests`。

### 4.3 常见错误码

- **401 Unauthorized**：API Key 无效或已过期。
- **402 Payment Required**：账户余额不足或订阅已过期。
- **503 Service Unavailable**：服务器维护中，通常发生在北京时间每周三凌晨 02:00-04:00。

## 5. 常见问题 (FAQ)

**Q: 我的数据会被用来训练你们的模型吗？**
A: 不会。根据 2023 年更新的用户协议，所有客户数据仅用于当次推理服务，存储周期不超过 30 天，且绝不用于基础模型训练。

**Q: 如何重置我的 API Key？**
A: 请登录控制台，进入 "开发者设置" 页面，点击 "重置密钥" 按钮。旧密钥将在 10 分钟后失效。

**Q: 支持钉钉或飞书集成吗？**
A: 目前支持飞书集成（需企业版），钉钉集成计划于 2024 年第四季度上线。

**Q: 如果我想取消订阅怎么办？**
A: 在 "账户设置" 中选择 "取消订阅"，服务将持续到当前计费周期结束。
